## 🚀 部署建议

### 开发环境
1. 启动后端服务：
   ```bash
   cd sirenkaifashiyong
   python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   ```

2. 访问前端页面：
   ```
   http://localhost:8000/static/auth.html
   ```
